# Automation Learning Projects

These are my automation projects built while learning automation using Zapier, Gmail, Google Forms and Google Sheets.

## Tools Used
- Zapier
- Gmail
- Google Forms
- Google Sheets
- AI Automation

## Projects

### 1. AI Inquiry Response Automation
Google Forms → Formatter → Paths (Fee Structure / Class Timing / Teacher Info) → Gmail → AI Analysis → Google Sheets → Email Reply

### 2. Email Auto Response System
Gmail Trigger → AI Analysis → Google Sheets → Send Email Reply

### 3. Lead Response Automation System
Google Forms → AI Analysis → Paths → Gmail Auto Response → Google Sheets Storage

### 4. AI Email Classification Automation
Gmail Trigger → AI Analysis → Paths (Inquiry / Complaint / Support) → Google Sheets → Email Reply

### 5. AI Social Media Post Generator
Google Forms → AI Content Generation → Google Sheets → Email Output

### 6. AI Business Email Automation
Gmail → AI Analysis → Paths (Yes / No) → Formatter → Google Sheets → Email Reply
